# Fire Fighter Drone On-Board Scripts
## Technologies
- Python
- PX4
- ROS
- OpenCV
- Dynamixel
## Functionality
- 30' prop
- 12s Lipo Battery
- 20" flight time
- 20 kg payload
- 40 km telemetry range
- Vision based target position estimation (using OpenCV)
- Dynamixel based fire extinguisher
